# ARVO - Otimização de Corte Têxtil

ARVO é um projeto acadêmico criado para ajudar no aproveitamento de tecido durante o processo de corte na confecção. O sistema organiza moldes automaticamente, reduz desperdícios e melhora o uso da matéria-prima.

Atualmente o projeto está na **Versão 3.0**.

---

## Evolução do Projeto

### Versão 1.0
Primeira base funcional.

- Entrada manual de medidas
- Inserção de moldes simples (retângulos)
- Geração inicial de layout
- Cálculo de área usada e desperdício

Foi a versão usada para validar a ideia inicial.

---

### Versão 2.0
Melhoria visual e lógica.

- Interface personalizada
- Cores diferentes para cada peça
- Rotação automática de moldes
- Lista de sobras
- Melhor organização visual

Foi a versão que transformou o projeto em algo mais profissional.

---

### Versão 3.0
Versão atual.

- Algoritmo inteligente de encaixe
- Teste de múltiplos layouts
- Escolha automática da melhor opção
- Peças em formato mais próximo de moldes reais
- Melhor aproveitamento do tecido
- Visualização das áreas sobrando

Essa versão representa a proposta principal do TCC.

---

## Objetivo

Mostrar como a tecnologia pode ajudar a indústria têxtil a economizar material, tempo e custos através da automação do corte.
